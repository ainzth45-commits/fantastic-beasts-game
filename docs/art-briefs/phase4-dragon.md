> ⏸WAIT: เริ่มได้เมื่อจิ้งจอกเมฆ (phase4-cloudfox.md) จบ + Codex ว่าง · ส่งตามลำดับรอบ

> **เอกลักษณ์มังกรชมพู (ทุกภาพต้องคงไว้):** เกล็ดสีชมพู ท้อง/อกครีม แผงหงอน-ผมสีม่วงพาสเทล เขาม่วงโค้งคู่ ปีกมังกรสีม่วง-ชมพูมีประกาย หางเกล็ดปลายแหลมมีหนามม่วงนุ่ม ตาม่วงโตประกาย แก้มชมพู — reference หลักทุกรอบ: `app/public/assets/concepts/concept-1-dragon.png`

## รอบ D1 — ชุดไข่ 6 ภาพ
```
เริ่มสัตว์ตัวที่ 3 (ตัวสุดท้าย) ของเกม Fantastic Beasts: "มังกรชมพู" — reference: เปิด app/public/assets/concepts/concept-1-dragon.png (ลูกมังกรชมพู แผงม่วง ปีกม่วง-ชมพู)
ออกแบบ "ไข่มังกรชมพู": เปลือกสีชมพูไล่ม่วงพาสเทล ลายเกล็ดหกเหลี่ยมนูนนุ่มๆ + หัวใจเล็กๆ ประดับ มีหนามม่วงนุ่มจิ๋ว 2-3 อันบนยอดไข่ วางบนปุยเมฆชมพูอ่อน (คนละใบกับไข่แกะ/จิ้งจอกชัดเจน)

สร้าง 6 ภาพ PNG ลง app/public/assets/beast-hard/ ชื่อเป๊ะๆ:
1. egg-idle.png — ไข่ชมพูนิ่งบนเมฆ ประกายรอบเบาๆ
2. egg-happy.png — ไข่สั่นดุ๊กดิ๊ก ประกาย+หัวใจลอยเยอะ
3. egg-sleep.png — ไข่หลับ ผ้าห่มเมฆคลุมครึ่งใบ ดาว zzz แสงสลัว (โปร่งใส ห้ามวงมืด)
4. cracking-idle.png — รอยแตกซิกแซกธรรมชาติ 2-3 เส้น จากรอยกว้างสุดมี "ปลายหางเกล็ดชมพูจิ๋วกับปุยแผงม่วง" โผล่นิดเดียว (ห้ามรูเหลี่ยมเข้ม ห้ามลูกบอลลอย)
5. cracking-happy.png — ไข่ร้าวสั่นดีใจ ประกาย+หัวใจ
6. cracking-sleep.png — ไข่ร้าวหลับ ผ้าห่มเมฆ zzz
สเปก: โปร่งใส PNG 1024x1024 · chibi พาสเทล · ห้ามตัวหนังสือ
เสร็จแล้วตอบว่าครบ 6 ไฟล์ไหม
```

## รอบ D8 — grown + adult 10 ภาพ ⏸รอ D7 ผ่าน
```
มังกรชมพู 2 ร่างสุดท้าย: "วัยหนุ่มสาว" (grown) และ "โตเต็มวัย" (adult)
reference: beast-hard/teen-idle.png + concept-1-dragon.png

grown: ปีกเต็มกางสง่า เขาม่วงเต็มความยาว เกล็ดเงาสวย ยังไม่มีออร่า
สร้าง: grown-idle / grown-blink / grown-happy / grown-sad / grown-sleep

adult (ร่างสุดท้าย อลังที่สุดใน 3 ตัว — เป็นสัตว์ระดับ hard): ปีกกางเต็มมีลายแสงรุ้งบนพังผืด "ออร่าเปลวเวทมนตร์ชมพู-ทอง" รอบตัว ประกายดาว+เกล็ดแสงลอย ยืนบนเมฆใหญ่ ท่าภูมิฐานราชามังกรแต่หน้ายังอ่อนโยนน่ารัก
สร้าง: adult-idle / adult-blink / adult-happy (พ่นประกายรุ้งเล็กๆ ฉลอง) / adult-sad / adult-sleep — ลง app/public/assets/beast-hard/
สเปกทุกภาพ: โปร่งใส 1024x1024 · chibi พาสเทล · ห้ามตัวหนังสือ · ตัวใหญ่เต็มเฟรมกว่าร่างก่อนเล็กน้อย
เสร็จแล้วตอบว่าครบ 10 ไฟล์ไหม
```
