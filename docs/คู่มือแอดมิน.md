# คู่มือแอดมิน — Fantastic Beasts (1 หน้า)

เว็บเกม: **https://ainzth45-commits.github.io/fantastic-beasts-game/**

## เปิดจอตอนเช้า
1. เปิดเบราว์เซอร์บนทีวี → เข้าเว็บ (ทำ bookmark ไว้) → กด F11 ให้เต็มจอ
2. รอโลโก้เข้ามุม → เจอการ์ด "น้อง...หลับอยู่" → กด **"เลี้ยงต่อ!"**
3. เช็คมุมขวาบน: ไม่มีป้าย ⚠️ = ยอดขายวิ่งเข้าเกมแล้ว ✅

> ⚠️ **จอเปิดใหม่จะ "พัก" เสมอ — ถ้าไม่กดเลี้ยงต่อ ยอดจะไม่นับ!** นี่คือหน้าที่สำคัญที่สุดของแอดมิน

## ระหว่างวัน
- ยอดขายเข้า CRM → เด้งเข้าเกมเอง ไม่ต้องทำอะไร (มีรูป+ชื่อคนขายเด้งบนจอ)
- **พักเที่ยง/ประชุม ไม่ต้องกดอะไร** — ปล่อยจอไว้ได้ น้องคิดยอดต่อเอง
- ปุ่ม 🔊/🔇 มุมขวาบน = เปิด/ปิดเสียง (จำค่าไว้ให้)
- กดโลโก้มุมซ้ายบน = กลับหน้าแรก (น้องพัก ยอดไม่นับ) → กดโลโก้ใหญ่ = กลับมาเลี้ยงต่อ

## เหตุการณ์พิเศษ
| เหตุการณ์ | ทำอะไร |
|---|---|
| น้องฟักออกจากไข่ 🐣 | จอเด้งให้ตั้งชื่อ — ให้ทีมช่วยกันคิด พิมพ์แล้วกด "ตั้งชื่อ 💜" (แก้ทีหลังได้ที่ปุ่ม ✏️) |
| น้องโตเต็มวัย 🎉 | จอฉลองใหญ่ + ขึ้นรางวัล — ประกาศรางวัลกับทีม แล้วกด **"รับรางวัล & เริ่มตัวถัดไป"** (ตัวใหม่เริ่มเป็นไข่ทันที) |
| อยากฉลองต่อก่อน | กด "ฉลองต่ออีกหน่อย" — จะมีป้าย 🎉 มุมขวาล่างไว้กดกลับมารับรางวัล |

## แก้ปัญหา
| อาการ | วิธีแก้ |
|---|---|
| ป้าย "⚠️ เชื่อมต่อยอดขายไม่ได้" | เช็คเน็ต → กด "พักเลี้ยง" แล้วกด "เลี้ยงต่อ" ใหม่ |
| จอค้าง/เพี้ยน | รีเฟรช (F5) → กด "เลี้ยงต่อ" — **แต้มไม่หาย** (เก็บในเครื่อง) |
| ยอดเด้งซ้ำ/ยอดถูกแก้ใน CRM | ไม่ต้องทำอะไร — เกมกันซ้ำ+ปรับแต้มตามอัตโนมัติ |
| ก่อนล้างเครื่อง/เปลี่ยนเครื่อง | เปิด `?dev=1` ท้าย URL → แผงเดโม → "📤 สำรองแต้ม" เก็บไฟล์ไว้ → เครื่องใหม่ "📥 กู้คืนแต้ม" |

## ห้ามทำ
- ❌ ห้ามกด "♻️ รีเซ็ตเกม" ในแผงเดโม (แต้มทีมหายทั้งเดือน!) — แผงนี้โผล่เฉพาะ `?dev=1`
- ❌ ห้ามล้าง cache/ข้อมูลเว็บของเบราว์เซอร์ทีวีโดยไม่สำรองแต้มก่อน
